== Description ==

Ce plugin gère votre calendrier de course, les résultats et vos RP.

== Installation ==

1. Envoyer le plugin dans le répertoire `/wp-content/plugins/agenda-running`, ou l'installer via l'écran d'administration de WordPress.
2. Activer le plugin via le menu 'Plugins'
3. Utiliser le panneau "AgendaRunning" pour ajouter vos courses
4. Ajouter le widget de votre choix (Agenda, Résultats, RP)
5. Personnalise en utilisant l'affichae personnalisé
